# node-red-contrib-dandelion
[![Build status][travis-image]][travis-url] [![NPM version][npm-image]][npm-url]

A node for [Node-RED](http://nodered.org) useful connect [dandelion](http://dandelion.eu) with API.

## Install
Use npm to command to install this package locally in the Node-RED modules directory
```bash
npm install node-red-contrib-dandelion 
```
or install in it globally with the command
```bash
npm install npm install node-red-contrib-dandelion -g 
```

## Nodes included in the package
**Dand_simil** check similarities between two fragments of text provided,
**Dand_extra** extract keywords from a text, 
**Dand_class** classify a fragment of test in different topics,
**Dand_detec** detect the language of a text fragment,
**Dand_senti** guess the sentiment of a text provided (WIP).
